*Versioning note:* OWASP materials and community summaries sometimes present minor naming/numbering differences across versions (e.g., some sources list Improper Output Handling as LLM05; others as LLM02; “Unbounded Consumption” is also described as Model DoS in some lists). We follow sponsor‑approved aliases and cross‑reference the official OWASP pages and PDF.
